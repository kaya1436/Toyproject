# Frontend

## Installation

- Node.js

```sh
https://nodejs.org/en/
```

```sh
$ node --version
v16.14.2
```

- Clone this repo

```sh
$ git clone https://github.com/kaya1436/Toyproject.git
```

- Install project dependancies

```sh
$ npm install
```

```sh
$ npm --version
v8.5.0
```

- start local server

```sh
$ npm run start
```
